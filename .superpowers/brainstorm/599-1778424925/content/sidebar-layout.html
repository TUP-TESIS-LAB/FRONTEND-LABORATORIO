¿Cómo debe comportarse el sidebar?

El color vendrá del tenant (white-label). La estructura es lo que definimos ahora.

A — Sidebar fijo expandido

Siempre visible con ícono + etiqueta de texto. Ancho fijo ~220px. Simple, predecible. Ideal para usuarios que aprenden el sistema.

B — Sidebar colapsable (íconos + expand)

Por defecto muestra solo íconos (~60px). Al hacer hover o click se expande mostrando etiquetas. Maximiza espacio de contenido.

C — Sidebar con grupos y secciones

Items agrupados por categoría (Core / Módulos opcionales / Admin). Con separadores y etiquetas de sección. Más organizado, mejor para muchos módulos.
